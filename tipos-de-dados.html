Tipos de Dados


Primeiramente oque é um dado?

Dado é a materia prima da informação. De forma isolada não traz conhecimento, não faz sentido.

Exemplo: 30

Informação

Já a informação podemos dizer que é um dado, que foi tratado, analizado e agora traz um siginificado.

Exemplo: Idade 30 anos.

Instrução Primitiva

A instrução é um conjunto de palavras-chaves (vocabulário) de uma linguagem de programação para o processamento de dados.

Exemplo: No Portugol temos um comando chamado escreva() que corresponde a uma ação que o algoritmo escrito na linguagem irá realizar.

Resumindo, dados são utilizados no contexto da computação para serem tratados (processados), gerando assim uma informação ou resolver um determinado problema.

Quais são os tipos de dados que iremos utilizar?

Dados Numéricos - Inteiros

Tipo de dados numéricos positivos e negativos (exceto fracionários).

Exemplo: 33 0 -128

Dados Numéricos - Reais

Tipo de dados numéricos positivos, negativos e fracionários.

Exemplos: 33.7 0.0 -12.9 5.1

Dados Literais

Sequência contendo letras, números e símbolos especiais. Uma sequência de Caracteres deve sempre ser indicada entre aspas duplas “ ”.

Exemplo: "Casa", "123", "num_apartamento", "Esta é uma frase formada por caracteres", " ".

Dados Lógicos

Dados com valores verdadeiro ou falso.

Exemplo: Se (condição) Verdadeiro - Faça isso. Senão Falso - Faça aquilo.
