Conceitos básicos necessários no desenvolvimento de algoritmos

Desenvolver o raciocinio lógico

Não basta apenas copiar e prestar a atenção em aulas. Para desenvolver raciocínio lógico, é necessário muita prática com questões e exercícios que farão com que você utilize a lógica e consiga desenvolver o seu raciocínio lógico, por isso você deverá resolver problemas.

Existem alguns passos necessários para resolver um problema, são eles:

  1. Conhecer as regras;
  2. Compreender o problema proposto;
  3. Indicar o que deve ser feito e em que ordem;
  4. Executar a sequência de passos e verificar se estes passos realmente resolvem o problema proposto

Um ótimo e simples exemplo é atravessar a rua, onde temos um problema que é atravessar a rua, e temos uma regra que é passar de uma calçada para outra sem ser atropelado

Sequência de ações que resolve o problema:

  1. Olhar para a esquerda
  2. Olhar para a direita
  3. Se estiver vindo carro - espere
  4. Repita passo 1 e 2
  5. Senão estiver vindo carro - atravesse

Raramente um problema terá apenas uma solução correta. Uma solução pode estar correta, mesmo sendo muito diferente da outra.

Algoritmo

O que é um algoritmo?

Sequência de instruções que resolve determinado problema.

Lógica de Programação

O que é lógica de programação?

Conjunto de raciocínios utilizados para criar um algoritmo. Como já sabemos, algoritmos é uma sequência de instruções que resolve um determinado problema.

O que é um programa?

É um algoritmo escrito em uma linguagem de programação específica, isto é, um algoritmo que pode ser executado em um computador.

Formas de representação de um algoritmo

Basicamente temos quatro formas de representação de um algoritmo.

  • Descrição Narrativa;
  • Fluxograma;
  • Diagrama de Chapin;
  • Pseudocódigo.

Todas essas formas de representações estão melhor detalhadas no material utilizado nas aulas, consulte os slides para aprofundar mais no conteúdo visto em sala.
